# QE Intelligence Engine (`qe-intelligence/`)

## What is the use of this folder?
This folder contains the Python-based AI and Analytics layer of the Quality Engineering platform.

## What it will do
It runs custom analytics scripts (like `metrics_aggregator.py` and `local_stress_runner.py`) that parse XML/JSON reports generated by the Java test suites. It tracks flaky rates, projects Test Impact Analysis (TIA) ROI, and validates false-positive/false-negative (FP/FN) ratios to ensure the test suite is mathematically reliable.

## How to use it
- Run the python scripts located in `scripts/` using a Python 3.10+ environment.
- E.g., `python scripts/metrics_aggregator.py`.
- Review the output data stored in the `data/` subdirectory.
